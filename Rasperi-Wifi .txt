Turn a Raspberry Pi into a WiFi router:

Chuẩn bị: - Thẻ nhớ 16gb --> 32gb.
	  - Raspberry pi 3 B+.
	  - Phần mền: SD card or win32 disk imager.

Mờ giao thức interfacing options: SSH và VNC.

Set ip tĩnh cho raspberry: 192.168.13.4/24 - interface

Set pass: - sudo passwb root :utc2hcm

1. Install the necessary software:

sudo apt-get update -y
sudo apt-get upgrade -y
sudo apt-get install iw -y
sudo apt-get install isc-dhcp-server -y
sudo apt-get install hostapd -y
sudo apt-get install iptables -y

2. Configure DHCP 
sudo nano /etc/dhcp/dhcpd.conf 
	
subnet 10.10.0.0 netmask 255.255.255.0{
range 10.10.0.25 10.10.0.150;
option domain-name-servers 8.8.4.4;
option routers 10.10.0.1;
interface wlan0;
}

3. Configure Hostapd
sudo nano /etc/hostapd/hostapd.conf
	
interface=wlan0
driver=nl80211
ssid=group_12
hw_mode=g
channel=11
ignore_broadcast_ssid=0
wpa=0
wpa_passphrase=khoangoc
wpa_key_mgmt=WPA-PSK
wpa_pairwise=TKIP
rsn_pairwise=CCMP
macaddr_acl=0

4. Configure IP static 
sudo nano /etc/network/interfaces

auto lo
iface lo inet loopback

auto eth0
allow-hotplug eth0
iface eth0 inet manual

auto wlan0

5. Start Wifi
+ Set ip for wlan0
sudo ifconfig wlan0 10.10.0.1 netmask 255.255.255.0
+ Start DHCP server
sudo /etc/init.d/dhcpcd start
sudo /etc/init.d/dhcpcd status
+ Start Hostapd
sudo hostapd -d /etc/hostapd/hostapd.conf
+Reboot 
sudo reboot

6.Nat 
/etc/sysctl.conf
net.ipv4.ip_forward=1

sudo iptables -t nat -A  POSTROUTING -o eth0 -j MASQUERADE

7. Tự động restart wifi
sudo nano /etc/rc.local
#set static wlan0
sudo ifconfig wlan0 10.10.0.1 netmask 255.255.255.0
#start dhcp server
sudo /etc/init.d/dhcpcd start
#set nat
sudo iptables -t nat -A  POSTROUTING -o eth0 -j MASQUERADE
#start hostapd
sudo hostapd -d /etc/hostapd/hostapd.conf
